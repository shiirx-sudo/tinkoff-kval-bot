# T-Invest Qualification Tracker

Инструмент для расчёта **прогресса к статусу квалифицированного инвестора**
в Т-Инвестициях по критерию **оборота сделок** за последние 4 завершённых квартала.

> **Read-only.** Используется только токен на чтение. Торговые заявки не отправляются
> (`LIVE_ENABLED=false` принудительно на Этапе 1).

## Что считает

- Оборот по покупкам/продажам ценных бумаг, фьючерсов и опционов.
- Исключает валюту, драгметаллы, комиссии, налоги, дивиденды, купоны,
  ввод/вывод средств, РЕПО и овернайты.
- Точный оборот по `trades` (`price × quantity`), с fallback на `payment`,
  если у операции нет детализации сделок (такие операции помечаются как
  приближённые и требуют сверки с брокерским отчётом).
- Разбивку по счетам и по кварталам, процент выполнения цели и остаток до неё
  (с учётом буфера безопасности).

## Расчёт периода

Период = ровно **4 последних завершённых квартала**. Примеры:

| Дата расчёта | Период                    |
|--------------|---------------------------|
| 2026-06-11   | 2025-04-01 … 2026-03-31   |
| 2026-01-15   | 2024-10-01 … 2025-12-31   |
| 2026-03-31   | 2024-10-01 … 2025-12-31   |
| 2026-04-01   | 2025-01-01 … 2025-12-31   |

## Установка

```bash
git clone https://github.com/shiirx-sudo/tinkoff-kval-bot.git
cd tinkoff-kval-bot
python -m venv .venv && source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env        # затем впишите токен в .env
```

В `.env` укажите **read-only** токен T-Invest API:

```env
TINKOFF_READ_TOKEN=ваш_токен_только_на_чтение
LIVE_ENABLED=false
```

## Запуск

```bash
python main.py                          # расчёт на сегодня, вывод в консоль
python main.py --as-of 2026-03-31       # на конкретную дату
python main.py --json out/report.json   # + JSON-отчёт
python main.py --csv out/trades.csv     # + CSV со сделками для сверки
python main.py -v                       # подробное логирование
```

## Безопасность

- Файл `.env` с токеном **не коммитится** (см. `.gitignore`). В репозиторий
  попадает только `.env.example` с плейсхолдером.
- Клиент реализует исключительно GET-методы; методы записи отсутствуют намеренно.

## Дисклеймер

Инструмент носит **информационный** характер и не является налоговой,
юридической или инвестиционной консультацией. Итоговые цифры всегда сверяйте
с официальным брокерским отчётом Т-Инвестиций. Пороговые значения и правила
учёта оборота могут меняться — проверяйте актуальные условия у брокера.
